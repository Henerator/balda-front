Правила игры

Лингвистическая игра, в которой необходимо составлять слова с помощью букв, добавляемых на игровое поле
За каждое слово игрок получает столько очков, сколько букв содержит это слово
Принимаются существительные в единственном числе
Не принимаются уже написанные ранее слова
Чтобы стать победителем наберите больше очков, чем ваш оппонент
